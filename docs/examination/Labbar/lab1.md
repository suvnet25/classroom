## Problemlösning och debugging

**Datum:** 2024-09-11

## Uppgift

I denna labb kommer du att skriva ett lite mindre program. Du kommer att behöva använda dig av:

* Variabler, if-satser, loopar
* Metoder
* Listor/Arrayer
* Inmatning och utskrift
* String-klassen och dess olika metoder

!!! tip "Använd debuggern för att felsöka ditt program eller för att förstå hur det fungerar!"
